Module org 

Source
Expand description

Organization membership workflows: add, revoke, update, and list members.

All workflows accept an OrgContext carrying injected infrastructure adapters (registry, clock, signer, passphrase provider). The CLI constructs this context at the presentation boundary; tests inject fakes.

Module key_material

Ed25519 key material parsing from various serialization formats.

Extracts SecureSeed (and optionally the public key) from PKCS#8 v1, v2, raw 32-byte seeds, and OCTET-STRING-wrapped seeds — pure byte parsing with no backend dependency.

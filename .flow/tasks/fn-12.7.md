# fn-12.7 auths-infra-git: add test-utils feature

## Description

Add a `test-utils` feature to `auths-infra-git` that exposes Git repository test helpers.

### What to move

From `crates/auths-test-utils/src/git.rs`:
- `init_test_repo() -> (TempDir, git2::Repository)` — creates a new temp dir + initialized git repo
- `get_cloned_test_repo() -> TempDir` — cloned copy of a shared template repo (OnceLock pattern)
- `copy_directory(src, dst)` — recursive directory copy helper

### Where to put it

Create `crates/auths-infra-git/src/testing.rs`.

Gate in `crates/auths-infra-git/src/lib.rs`:
```rust
#[cfg(any(test, feature = "test-utils"))]
pub mod testing;
```

### Cargo.toml changes

```toml

test-utils = ["dep:tempfile"]
```

`tempfile` must be an optional normal dependency (not dev-dep) since other crates will use it via the feature.

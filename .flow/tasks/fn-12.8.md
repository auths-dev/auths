# fn-12.8 Migrate all consumers to per-crate test-utils

## Description

Update all crates that currently use `auths-test-utils` to instead use the per-crate `test-utils` features created in fn-12.1 through fn-12.7.

### Consumers to migrate

| Crate | Current import | New import |
|---|---|---|
| `auths-crypto` | `auths_test_utils::crypto::create_test_keypair` | `auths_crypto::testing::create_test_keypair` |
| `auths-verifier` (src/verify.rs, src/witness.rs, tests/) | `auths_test_utils::crypto::create_test_keypair` | `auths_crypto::testing::create_test_keypair` |
| `auths-core` | `auths-test-utils.workspace = true` | `auths-crypto = { workspace = true, features = ["test-utils"] }` |
| `auths-id` | `auths_test_utils::*` (fakes, fixtures, contracts, git) | `auths_id::testing::*` + `auths_infra_git::testing::*` |
| `auths-sdk` | `auths_test_utils::fakes::*` | `auths_id::testing::fakes::*` + `auths_sdk::testing::fakes::*` |
| `auths-infra-git` | `auths_test_utils::git::*`, `auths_test_utils::fakes::git::*` | `auths_infra_git::testing::*` + `auths_sdk::testing::fakes::*` |
| `auths-infra-http` | `auths-test-utils.workspace = true` | Appropriate per-crate features |
| `auths-cli` | `auths_test_utils::crypto::gen_keypair` | `auths_crypto::testing::gen_keypair` |

### For each crate

1. Replace `auths-test-utils.workspace = true` in `[dev-dependencies]` with the specific per-crate features needed
2. Update all `use auths_test_utils::*` import paths to the new locations
3. Run `cargo nextest run -p <crate>` to verify

### Important: auths-verifier WASM

`auths-verifier` has target-specific dev-dependencies. The `auths-crypto` test-utils dev-dep should be under `[target.'cfg(not(target_arch = "wasm32"))'.dev-dependencies]` to avoid WASM build issues.

### Files to modify

- `Cargo.toml` dev-dependencies in: `auths-crypto`, `auths-core`, `auths-verifier`, `auths-id`, `auths-sdk`, `auths-infra-git`, `auths-infra-http`, `auths-cli`
- All `use auths_test_utils::` import statements across the workspace
- `auths-verifier/src/verify.rs` (~70 uses in `#[cfg(test)]` block)
- `auths-verifier/src/witness.rs` (~12 uses in `#[cfg(test)]` block)

### Verification

```bash
cargo build --workspace
cargo nextest run --workspace
cargo clippy --all-targets --all-features -- -D warnings
```

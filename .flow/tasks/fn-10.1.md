## Add PublishError, PublishOutcome types and publish_artifact SDK workflow

### Context

`crates/auths-cli/src/commands/artifact/publish.rs` currently:
- Creates its own `reqwest::Client` (30s connect + 60s total timeout) at line 165 -- different from the shared CLI client
- POSTs JSON to `{registry_url}/v1/artifacts/publish` at line 170
- Dispatches on HTTP 201/409/422 status codes at lines 137-153
- Contains `validate_package_identifier()` (lines 40-58) -- pure business logic in the CLI
- Uses `anyhow::bail!` for error handling

### What to Do

**1. Add error and result types in `crates/auths-sdk/src/error.rs`:**

```rust
#[derive(Debug, thiserror::Error)]
#[non_exhaustive]
pub enum PublishError {
    #[error("attestation already published with this RID")]
    AlreadyPublished,

    #[error("registry rejected attestation: {detail}")]
    VerificationRejected { detail: String },

    #[error("invalid package identifier: {reason}")]
    InvalidPackageIdentifier { reason: String },

    #[error("invalid attestation file: {reason}")]
    InvalidAttestationFile { reason: String },

    #[error("network error")]
    NetworkError(#[source] NetworkError),
}
```

**2. Add `PublishOutcome` in `crates/auths-sdk/src/result.rs`** (or create it if absent):

```rust
pub struct PublishOutcome {
    pub message: String,
}
```

**3. Create `crates/auths-sdk/src/workflows/publish.rs`:**

Follow `registration.rs` pattern exactly:
- Function signature: `pub async fn publish_artifact(registry_client: &impl RegistryClient, attestation_json: &serde_json::Value, package_name: Option<&str>, registry_url: &str) -> Result<PublishOutcome, PublishError>`
- Move `validate_package_identifier()` logic here
- Build the request payload (same JSON structure as current `transmit_publish`)
- Call `registry_client.post_json(registry_url, "/v1/artifacts/publish", &payload_bytes)`
- Map `RegistryResponse.status`: 201 -> Ok, 409 -> AlreadyPublished, 422 -> VerificationRejected, other -> NetworkError
- No `println!`, no `eprintln!`, no `anyhow`, no `reqwest` imports

**4. Export from `crates/auths-sdk/src/lib.rs`**

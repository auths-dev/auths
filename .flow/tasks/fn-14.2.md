## Description
# Task: CESR Codec -- Encoding and Decoding Primitives

Implement the `CesrCodec` trait backed by `cesride`. This is the lowest layer -- it knows about CESR code tables but nothing about KERI events.

## What to do

1. Implement `CesrV1Codec::encode_pubkey` using `cesride::Verfer`:
   - `Verfer::new(Some(matter::Codex::Ed25519), key_bytes, None)` -> `.qb64()`
   - Ed25519: `D` + 43 chars base64url = 44 chars total

2. Implement `CesrV1Codec::encode_indexed_signature` using `cesride::Siger`:
   - **IMPORTANT**: cesride's Siger::new takes 8 Option params: `(verfer, index, ondex, code, raw, qb64b, qb64, qb2)`
   - For creating from raw bytes: `Siger::new(None, Some(key_index), None, None, Some(sig_bytes), None, None, None)`
   - Ed25519 indexed at index 0: `AA` + 86 chars = 88 chars total

3. Implement `CesrV1Codec::encode_digest` using `cesride::Diger`:
   - `Diger::new(None, Some(matter::Codex::Blake3_256), Some(digest_bytes), None)` -> `.qb64()`
   - Blake3-256: `E` + 43 chars = 44 chars total

4. Implement `CesrV1Codec::decode_qualified`:
   - Try `Verfer::new(None, &[], Some(qualified))` first (public keys)
   - Then `Diger::new(None, None, None, Some(qualified))` (digests)
   - Return `DecodedPrimitive { raw, code }`

5. Write tests in `tests/cases/codec.rs`:
   - 32 zero-bytes encodes to `"DAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAA"` (matches `KeriPublicKey::parse` test)
   - decode(encode(key)) returns original bytes
   - 64-byte signature at index 0 produces 88-char string starting with `AA`
   - 32-byte digest produces 44-char string starting with `E`
   - proptest round-trip: for any random 32-byte key, `decode(encode(key)) == key`

## cesride Siger API (corrected from original spec)

The Siger constructor signature is:
```rust
Siger::new(
    verfer: Option<&Verfer>,     // associated verification key
    index: Option<u32>,          // key index
    ondex: Option<u32>,          // other index
    code: Option<&str>,          // signature algorithm code
    raw: Option<&[u8]>,          // raw signature bytes
    qb64b: Option<&[u8]>,       // qualified base64 bytes
    qb64: Option<&str>,         // qualified base64 string
    qb2: Option<&[u8]>,         // qualified binary
)
```
